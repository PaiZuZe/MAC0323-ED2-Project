### Tools

+ **Wrapper**

    Generates a .tar.gz file for the given program (tests and object files won't
    be included)

    ```
    ./wrapper.sh <binary>
    ```

+ **Newdir**

    Creates a new directory parte_dir with a Makefile template

    ```
    ./newdir.sh <dir>
    ```
